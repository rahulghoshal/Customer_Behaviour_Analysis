# Customer Behavior Analysis – Data Analytics Project

## Overview
This project focuses on analyzing customer purchasing behavior using a complete data analytics workflow. The analysis includes loading and preprocessing a dataset in Python, performing Exploratory Data Analysis (EDA), cleaning the data, executing analytical queries in MySQL Server, and creating an interactive Power BI dashboard.  

The goal of the project is to identify customer purchase patterns, understand the effect of discounts on buying behavior, and generate insights that can support better business and marketing decisions.

---

## Dataset
The dataset contains customer purchase and behavioral information, including:

- Customer ID  
- Age  
- Gender  
- Purchase Amount  
- Previous Purchases  
- Discount Applied  
- Category  
- Review Rating  

This data helps analyze spending patterns, customer segments, and purchasing trends.

---

## Tools & Technologies
- **Python**
  - Pandas
- **SQL**
  - MySQL Server
- **Data Visualization**
  - Power BI

---

## Project Steps

### 1. Data Loading
- Imported the dataset using **Python (Pandas)**.
- Inspected the structure of the dataset and checked data types.

### 2. Data Cleaning
- Handled missing values.
- Removed inconsistencies and duplicate records.
- Standardized column formats for analysis.

### 3. Exploratory Data Analysis (EDA)
- Analyzed purchase distribution.
- Studied the relationship between discounts and purchase amount.
- Identified patterns in customer buying behavior.
- Visualized trends using Python charts.

### 4. SQL Analysis
- Loaded the cleaned dataset into **MySQL Server**.
- Wrote SQL queries to:
  - Identify high-spending customers
  - Analyze discount usage
  - Segment customers based on previous purchases
  - Calculate average purchase metrics

### 5. Data Visualization
- Connected the processed dataset to **Power BI**.
- Created interactive dashboards to visualize customer insights.

---

## Dashboard
The Power BI dashboard includes:

- Customer segmentation analysis  
- Purchase amount distribution  
- Discount usage insights  
- Category-wise sales analysis  
- Key performance indicators (KPIs)

The dashboard helps stakeholders quickly understand customer behavior and business trends.

---

## Results
Key insights obtained from the analysis include:

- Loyal customers contribute significantly to total revenue.
- Discount offers influence purchasing decisions.
- Certain product categories generate higher spending.
- Customer segmentation helps identify targeted marketing opportunities.
